# Navegantes do Capibaribe
Jogo de RPG 2D de aventura e exploração histórica · Recife Antigo & Rio Capibaribe
* Vibe recifense: Recife Antigo
* Gênero: RPG/Aventura
* Elemento central: Rio Capibaribe
* Público-alvo: Estudantes do ensino médio e superior com interesse em história local, jogadores casuais de RPG e turistas virtuais interessados na cultura pernambucana.
